# 什么是国际短信

## 产品概述

ASMS 短信服务（Abroad Short Message Service）是 UMCloud 提供的一种云通信服务，帮助广大用户拓展海外市场。

国际及港澳台短信覆盖全球 200+国家及地区。常见应用场景：验证码短信（账号注册、登录验证、找回密码、安全提醒等场景），系统通知短信（故障告警、状态提醒、价格调整等场景），会员服务短信（活动通知、业务促销、新品上线、会员关怀等场景）。
